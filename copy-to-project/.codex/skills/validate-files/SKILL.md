name: validate-files
description: Валидация .json и .md файлов после редактирования.
---

После каждого редактирования .json или .md файла выполни проверку:

## JSON

Убедись что JSON синтаксически корректен. Проверь: закрытые скобки, запятые, кавычки.

## Markdown

1. **Блоки кода** — каждый открывающий ``` имеет закрывающий ```
2. **Ссылки** — нет пустых ссылок `[текст]()`
3. **@imports** — файлы в `@path/to/file` существуют в проекте

## При обнаружении ошибки

Исправь проблему сразу, до продолжения работы.
